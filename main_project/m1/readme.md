Hey everyone. Use this folder to do your main project work.

Few Guidelines:
1. Before pushing your work, you must pull the current stage of this branch. Use command - git pull origin M1
2. Now Use these commands in order to push your work
    1. git add .
    2. git commit -m "Commit name"
    3. git push origin M1
    
    This will make sure that all the conflicts between your work and remote repo is resolved.
